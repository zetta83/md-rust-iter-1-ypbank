# ypbank_parser

Библиотека для парсинга и конвертации транзакционных записей в трёх форматах: CSV, TXT и бинарный (BIN).

## Форматы данных

- [YPBankBin](./YPBankBinFormat_ru.md)
- [CSV-формата YPBank](./YPBankCsvFormat_ru.md)
- [YPBankText Text File Format](./YPBankTextFromat_ru.md)

## О парсере

`ypbank_parser` — это универсальная библиотека для чтения, записи и конвертации транзакционных записей между тремя форматами: CSV, TXT и бинарным. Библиотека реализует единый интерфейс `Parser` для всех форматов и предоставляет тип-обёртку `ParserByType`, позволяющий работать с разными форматами через единый API.

### Основные возможности

- **Чтение записей** из файлов в форматах CSV, TXT и BIN с потоковой обработкой (через реализацию `Iterator`)
- **Запись записей** в любой из поддерживаемых форматов
- **Конвертация между форматами** без промежуточного хранения всех данных в памяти
- **Сравнение файлов** разных форматов с детализированным выводом различий
- **Буферизированный ввод/вывод** для оптимальной производительности

### Особенности реализации

- **Единый трейт `Parser`** — все парсеры реализуют общий интерфейс с методами `read_from`, `write_record`, `write_header` и `convert_to`
- **Потоковая обработка** — записи обрабатываются по одной, что позволяет работать с файлами любого размера
- **Гибкое отображение полей** — CSV-парсер автоматически определяет индексы колонок по заголовку, поддерживая любой порядок полей
- **Валидация данных** — проверка магического числа для BIN-формата, обязательных полей и корректности типов
- **Детализированные ошибки** — все возможные ошибки парсинга типизированы и содержат контекст

### Поддерживаемые операции

| Из \ В | CSV | TXT | BIN |
|--------|-----|-----|-----|
| CSV    | ✅   | ✅   | ✅   |
| TXT    | ✅   | ✅   | ✅   |
| BIN    | ✅   | ✅   | ✅   |

### Требования

- Rust 1.70+
- Стандартная библиотека (no_std не поддерживается из-за использования `std::io`)

### Интеграция

Библиотека может использоваться как:
- **Библиотека** — для встраивания в другие Rust-проекты
- **CLI-утилиты** — для конвертации и сравнения файлов из командной строки
